# 🌦️ Weather Analysis & Prediction using Time Series Linear Regression

## 📌 Overview

Weather plays a critical role in sectors like agriculture, where accurate predictions directly impact productivity and decision-making. With rapidly changing climate patterns, traditional prediction methods are becoming less effective.

This project focuses on building a **low-cost, data-driven weather prediction system** using **Machine Learning (Linear Regression)** to forecast rainfall and precipitation levels.

---

## 🎯 Objective

* Develop a reliable and simple weather prediction model
* Enable predictions in **remote and resource-limited areas**
* Provide insights on rainfall intensity (Low, Moderate, Heavy)

---

## 🧠 Methodology

* Collected and analyzed historical weather data
* Applied **data preprocessing and feature engineering**
* Used **Linear Regression algorithm** for prediction
* Evaluated model performance using standard metrics

---

## 📊 Features Used

* Temperature
* Humidity
* Wind Speed
* Sea Level Pressure

---

## ⚙️ How It Works

1. User inputs current weather parameters
2. Model processes input data
3. Predicts:

   * Precipitation value
   * Rainfall category (Low / Moderate / Heavy)

---

## 🛠️ Tech Stack

---

## 📈 Output

* Predicted rainfall values
* Visualized trends and patterns
* Classification of rainfall intensity
